# Telegram HTML Crypto Bot

A Telegram bot for encrypting, decrypting, obfuscating, and deobfuscating HTML files.

## Features
- 🔐 Encrypt HTML files
- 🔓 Decrypt encrypted HTML
- 🔀 Obfuscate HTML code
- ✨ Deobfuscate HTML code

## Commands
- `/encrypt [html]` - Encrypt HTML
- `/decrypt [data]` - Decrypt HTML
- `/obfuscate [html]` - Obfuscate HTML
- `/deobfuscate [data]` - Deobfuscate HTML
